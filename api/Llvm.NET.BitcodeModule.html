Remarks
note

The data layout string doesn't do what seems obvious. That is, it doesn't force the target back-end to generate code or types with a particular layout. Rather, the layout string has to match the implicit layout of the target. Thus it should only come from the actual TargetMachine the code is targeting.

MakeShared()

Makes this module a shared module if it isn't already

Declaration
public void MakeShared()
Remarks

Normally, this is called automatically when adding a module to a JIT engine that needs shared modules. Shared modules are reference counted so they aren't completely disposed until the reference count is decremented to 0. Calling Dispose() on a shared module will mark the module as disposed and decrement the share count.
