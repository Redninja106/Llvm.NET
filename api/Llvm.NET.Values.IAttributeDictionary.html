Remarks

This interface provides a full collection of all the attributes keyed by the FunctionAttributeIndex

note

This conceptually corresponds to the functionality of the LLVM AttributeSet class for Versions prior to 5. In LLVM 5 the equivalent type is currently AttributeList. In v5 AttributeSet has no index and is therefore more properly a set than in the past. To help remove confusion and satisfy naming rules this is called a Dictionary as that reflects the use here and fits the direction of LLVM
