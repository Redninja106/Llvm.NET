Remarks

Ordinarily layout information is handle automatically in SetBody(Boolean, BitcodeModule, DIScope, DIFile, UInt32, DebugInfoFlags, IEnumerable<DebugMemberInfo>) however in cases where explicitly controlled (or "packed") layout is required, instances of DebugMemberLayout are used to provide the information necessary to generate a proper type and debug information.
