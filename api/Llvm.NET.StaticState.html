InitializeLLVM()

Initializes the native LLVM library support

Declaration
public static IDisposable InitializeLLVM()
Returns
Type Description
System.IDisposable

System.IDisposable implementation for the library
Remarks

This should be called once per application to initialize the LLVM library. System.IDisposable.Dispose() will release any resources allocated by the library.
