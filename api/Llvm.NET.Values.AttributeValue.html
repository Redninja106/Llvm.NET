public struct AttributeValue : IEquatable<AttributeValue>
Remarks

This is the equivalent to the underlying llvm::AttributeImpl class. The name was changed to AttributeValue in .NET to prevent confusion with the standard System.Attribute class that is used throughout .NET libraries.
