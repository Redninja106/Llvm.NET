Remarks

This class is used with DebugStructType to provide debug information for a type.

In order to support explicit layout structures the members relating to layout are all System.Nullable<T>. When they are null then modules Layout target specific layout information is used to determine layout details. Setting the layout members of this class to non-null will override that behavior to define explicit layout details.

Remarks

If this is null then SetBody(Boolean, BitcodeModule, DIScope, DIFile, UInt32, DebugInfoFlags, IEnumerable<DebugMemberInfo>) will default to using Layout to determine the size using the module's target specific layout.

note

If this property is provided (e.g. is not null) for any member of a type, then it must be set for all members. In other words explicit layout must be defined for all members or none. Furthermore, for types using explicit layout, the type containing this member must include the "packed" modifier.
